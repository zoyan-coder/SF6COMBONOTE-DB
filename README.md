# SF6 COMBO NOTE DB

Street Fighter 6 のコンボを記録・整理するための個人研究ノートツールです。

---

## コンセプト

- **個人のコンボ研究ノート**として使うことを想定しています
- 競技向けの分析ツールではなく、練習中に気づいたコンボを手軽にメモする感覚で使えます
- シングルHTMLファイルで動作します。インストール不要、サーバー不要

---

## 仕組みと注意事項

データは **ブラウザのIndexedDB**（ローカルストレージ）に保存されます。

| 注意点 | 対策 |
|--------|------|
| ブラウザのデータ削除でコンボが消える | 定期的にJSONエクスポートでバックアップ |
| 別ブラウザ・別PCではデータが引き継がれない | JSONをインポートして移行 |
| プライベートモードでは保存されない | 通常モードで使用 |

**環境を変えるときは必ずエクスポートしてから移行してください。**

## 使い方

### 基本

1. [こちら](https://zoyan-coder.github.io/SF6COMBONOTE-DB/sf6comboNoteDB.html) をブラウザで開く
2. **新規登録** でキャラクターを選択してコンボルートを入力
3. **マスタ管理** でキャラクター・タグ・ラベルを管理

### コンボルートの入力

- 右パレットのチップをクリック or ドラッグしてルートに追加
- 「＋」ボタンで技名検索・新規登録
- 各チップの「遅」で遅らせ入力、「AT」でオートボタン入力をマーク

### モダン操作

パレット下部の **⚡ モダン操作 ON/OFF** で切り替えると、通常技パレットがモダン仕様（方向＋L/M/H）に切り替わります。

### コンボの共有

- コンボカード右の **⬆** ボタンで1件エクスポート
- ツールバーの **⬆ 出力** でフィルタ中の全件をエクスポート
- **↓ 取り込む** で他の人のコンボを取り込み（タグは名前で自動マッチング）

### バックアップと移行

マスタ管理 → データ管理 からJSONエクスポート／インポートができます。

## 推奨環境

| 項目 | 推奨 |
|------|------|
| デバイス | PC（Windows / Mac） |
| ブラウザ | Chrome / Edge / Firefox 最新版 |
| 画面幅 | 1024px 以上 |

> スマートフォン・タブレットでは一部機能（ドラッグ＆ドロップ等）が正しく動作しない場合があります。
